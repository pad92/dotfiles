# Install Dotfile

```sh
git clone https://gitlab.com/pad92/dotfiles.git ~/.dotfiles
./.dotfiles/install
```

## Install only VIM

```sh
curl -sSL https://gitlab.com/pad92/dotfiles/-/raw/master/vim.sh | sh
```
