# Proton Pass Zsh Plugin

This plugin adds helper aliases for the official Proton Pass CLI (`pass-cli`).

## Aliases

| Alias | Command | Description |
|---|---|---|
| `proton-pass` | `pass-cli` | Call Proton Pass CLI |
| `proton-pass-item` | `pass-cli item` | Manage vault items |
| `proton-pass-vault` | `pass-cli vault` | Manage vaults |
| `proton-pass-totp` | `pass-cli totp` | Retrieve TOTP codes |
| `proton-pass-run` | `pass-cli run` | Run commands with secrets injected into environment |
| `proton-pass-agent` | `pass-cli ssh-agent` | Interact with the SSH agent |
| `proton-pass-agent-start` | `pass-cli ssh-agent daemon start` | Start the SSH agent background daemon |
| `proton-pass-agent-status` | `pass-cli ssh-agent daemon status` | Check the status of the daemon |
| `proton-pass-agent-stop` | `pass-cli ssh-agent daemon stop` | Stop the daemon |
